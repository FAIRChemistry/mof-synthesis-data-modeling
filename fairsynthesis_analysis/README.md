# Data analysis tools
## Running
1. Install [uv](https://github.com/astral-sh/uv) by e.g., 'pip install uv'
2. open cmd / terminal, go to `fairsynthesis_analysis` folder and type `uv run marimo edit`
3. Workspace > examples > pxrd_analysis.mo.py or correlation_analysis.mo.py
4. run all slate cells (right bottom)
