# Data

This folder contains two example datasets.

## [Synthesis of MOCOF-1 based on Sciformation electronic lab notebook](./MOCOF-1)

This folder contains the dataset for [the synthesis of MOCOF-1](https://www.nature.com/articles/s44160-024-00719-x).
The synthetic procedures and product masses were exported from Sciformation electronic lab notebook, which has been cleaned and converted to the MOFSY format.

## [Synthesis of Fe–terephthalate based on CSV](./Fe-terephthalate)

This folder contains the dataset for the synthesis of Fe–terephthalate.
The synthetic procedures and product masses were recorded in the CSV format, which has been converted to MOFSY format.
